# AIE1001 期中复习报告（中文版）

## 引言
这份报告总结了 AIE1001 人工智能编程导论课程第 1 周至第 6 周的关键概念和常见易错点。期中考试涵盖这些主题，重点包括 Python 基础、控制结构、函数、数据结构和递归。理解这些基础知识对于使用 Python 编程 AI 应用至关重要。

## 第 1 周：人工智能编程导论
### 重点内容
- **计算机基础**：冯·诺伊曼架构、CPU、内存（RAM/ROM）、输入/输出设备。
- **AI 影响**：历史革命、AI 应用（如 AlphaFold、自动驾驶汽车）、局限性（鲁棒性、数据依赖）。
- **编程语言**：从汇编到 Python 的演变；效率 vs. 开发速度。
- **数制**：二进制、十进制、八进制、十六进制转换；位置记数法；单位（位、字节、KB 等）。
- **数据表示**：内存寻址、抽象层（硬件到应用）。

### 常见易错点
- 数制转换中小数部分的错误（例如，十进制转二进制乘以 2）。
- 转换基数混淆（例如，忘记为八进制/二进制填充零）。
- 误解二进制使用原因（物理实现）及其与十六进制的关联（分组 4 位）。
- 忽略变量引用内存位置，而非直接存储值。

## 第 2 周：Python 基础
### 重点内容
- **解释器 vs. 编译器**：Python 使用解释器，支持交互/脚本模式。
- **变量和数据类型**：命名规则、常量（大写）、int/float/str、type() 函数。
- **类型转换**：int()、float()、str()；混合表达式中的隐式转换。
- **表达式和运算符**：算术运算（+、-、*、/、//、%、**）、优先级（PEMDAS）、增量赋值（+= 等）。
- **输入/输出**：input() 返回 str；print() 用于输出；eval() 用于动态求值。
- **字符串**：连接（+、*）、索引。

### 常见易错点
- 忘记 input() 返回字符串，导致算术类型错误（例如，'2' + 3）。
- 运算符优先级错误（例如，1 + 2 * 3 = 7，而非 9）。
- 脚本中的缩进错误。
- 使用 == 进行赋值而非 =。
- while 循环条件错误导致无限循环。
- eval() 与不安全输入的误用。

## 第 3 周：流程控制
### 重点内容
- **比较**：==、!=、<、>、<=、>=；字符串字典序比较（ASCII/Unicode）。
- **布尔**：True/False、bool()、逻辑运算符（and、or、not）。
- **条件语句**：if、elif、else；嵌套 if；多路决策。
- **循环**：while（不定循环）、for（定循环）结合 range()；break、continue。
- **异常处理**：try/except 处理如 ValueError 的错误。
- **浮点数**：比较中的精度问题。

### 常见易错点
- 浮点精度（例如，0.1 + 0.2 != 0.3 由于近似）。
- 字符串比较的大小写敏感；忽略非 ASCII 字符的 Unicode。
- 循环条件中的越界错误（例如，while i < len(s) vs. <=）。
- 缩进不一致导致语法错误。
- 忘记 else 或 elif 位置错误。
- 缺少适当终止条件导致无限循环。

## 第 4 周：函数
### 重点内容
- **定义**：def、参数 vs. 参数、return vs. 无返回（None）。
- **作用域**：局部 vs. 全局变量；谨慎使用 global 关键字。
- **高级**：默认参数、多重返回（元组）、嵌套调用。
- **有返回函数**：返回值为复用；无返回用于副作用（例如，print）。

### 常见易错点
- 调用时参数顺序不匹配。
- 作用域错误：修改局部变量不影响全局；意外使用全局。
- 忘记 return 语句，导致返回 None。
- 递归深度超限（尽管基本递归在后介绍）。
- 默认参数在定义时求值，而非调用（可变默认参数陷阱）。
- 过度使用全局变量，导致调试困难。

## 第 5 周：列表、字典和元组
### 重点内容
- **列表**：可变序列 [ ]、索引/切片、len()、append()、sort()、in 运算符；内置函数（max、min、sum）。
- **字典**：{key: value}、无序、get()、keys()/values()/items()；使用字典计数。
- **元组**：不可变 ()、固定数据高效；用于排序的可比性。
- **操作**：连接（+）、重复（*）、split() 将字符串转列表。
- **排序**：list.sort()（原地）vs. sorted()；按键/值排序字典。

### 常见易错点
- 索引越界（列表/字符串从 0 开始，len() 排除最后一个索引）。
- 修改元组（不可变错误）。
- 字典访问的 KeyError；忘记 get() 用于默认值。
- 假设字典顺序（Python 3.7+ 保持插入顺序，但不保证）。
- 列表方法混淆（例如，sort() vs. sorted()）。
- 构建列表时未正确使用 append() 导致无限循环。

## 第 6 周：递归
### 重点内容
- **递归基础**：基例（无调用）、递归例（用更小输入调用自身）。
- **示例**：阶乘（n! = n * (n-1)!）、幂（x^n）、二分搜索（有序列表）。
- **线性递归**：一个递归调用（例如，求列表和）。
- **多重递归**：多个调用（例如，斐波那契、英式尺子）。
- **效率**：调用栈帧；尾递归优化（Python 不支持）。

### 常见易错点
- 缺少或不正确的基例，导致无限递归（例如，fact(0) 未处理）。
- 递归条件中的越界（例如，n==1 时 n-1）。
- 深递归导致栈溢出（Python 限制 ~1000 调用）。
- 误解调用栈：每个调用有自己的帧，返回时展开。
- 多重递归低效（例如，斐波那契重复计算值）。
- 忘记在递归例中缩小问题规模。

## 结论
专注于练习代码执行（可使用 Python Tutor 可视化递归和循环）。常见错误往往源于类型、作用域和越界索引。复习数制和浮点数边缘情况。考试中，强调编写正确的 Python 代码，包括函数、循环和递归解决方案。祝好运！
